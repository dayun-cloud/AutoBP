# AutoBP - 英雄联盟自动BP助手

一个功能强大的英雄联盟自动Ban/Pick助手，支持自动接受对局、智能预选英雄、自动禁用英雄和按位置自动选择英雄等功能。

📥 **[点击下载最新版本（Go版本）](https://github.com/dayun-cloud/AutoBP/releases/latest/download/AutoBP.exe)**

🚀 **[查看Go版本文档](AutoBP-Go/README-Go.md)** - 基于Wails框架的现代化桌面应用版本

## ✨ 主要功能

### 🎯 核心功能
- **自动接受对局** - 自动接受匹配到的游戏
- **智能预选英雄** - 进入选人阶段后自动显示意向英雄
- **自动禁用英雄** - 在Ban阶段自动禁用指定英雄
- **自动选择英雄** - 在Pick阶段自动锁定英雄

### 🎮 按位置选择功能
- **智能位置识别** - 自动识别玩家在排位中的分配位置
- **按位置配置英雄** - 为每个位置（上路、打野、中路、下路、辅助）单独配置英雄

### 🖥️ 用户界面
- **现代化Web界面** - 简洁美观的用户界面
- **实时状态显示** - 显示LCU连接状态和客户端状态
- **英雄搜索功能** - 支持英雄名称搜索和快速选择

## 🚀 快速开始

### 环境要求
- Python 3.7+
- 英雄联盟客户端

### 安装依赖
```bash
pip install -r requirements.txt
```

### 运行程序
```bash
python main.py
```

## 🛠️ 技术架构

### 后端技术
- **FastAPI** - 现代化的Python Web框架
- **lcu-driver** - 英雄联盟客户端API连接器
- **asyncio** - 异步编程支持
- **httpx** - 异步HTTP客户端

### 前端技术
- **原生JavaScript** - 无框架依赖
- **现代CSS** - 响应式设计和暗色主题
- **实时通信** - 与后端API实时交互

### 核心组件
- **LCU连接器** - 监听游戏客户端事件
- **配置管理** - 动态加载和保存配置
- **英雄数据管理** - 自动更新英雄信息
- **Web服务** - 提供用户界面和API接口

## 📦 打包部署

使用PyInstaller打包为独立可执行文件：

```bash
pyinstaller AutoBP.spec
```

打包后的文件位于 `dist` 目录中，可以直接运行无需安装Python环境。

## ⚠️ 注意事项

1. **客户端要求** - 需要英雄联盟客户端运行并登录
2. **权限要求** - 程序需要访问LCU API（本地端口）
3. **网络要求** - 首次运行需要网络连接以获取英雄数据
4. **使用规范** - 请遵守游戏规则，合理使用自动化功能
